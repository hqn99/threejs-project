# 3D ROOM
プレ卒業制作にて、Three.jsを用いてWebブラウザ上で3D空間を探索できるシステムを構築しました。

## サイトURL
https://threejs-project-cyan.vercel.app/

## 制作期間
- 開始日：2025年10月23日
- 終了日：2026年2月9日

## 使用ツール
- モデリング： Maya
- 描画・制御： Three.js (JavaScript)

## 操作方法
左クリック長押しかつカーソル移動で視点変更
右クリックでアクション
